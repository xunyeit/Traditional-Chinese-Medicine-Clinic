## 一、产品概述
**“****仁医星中医馆****”是一款专为现代人打造的****数字化中医健康管理平台****，融合传统中医药理念与现代移动互联网技术，致力于提供“****线上咨询 + 健康记录 + 中医科普 + 个性化诊疗服务****”的一站式健康管理解决方案。该应用面向关注亚健康、慢性病管理、体重控制、体质调理等人群，尤其适合需要长期跟踪血糖、血压、饮食、用药情况的患者群体。**

**平台以“****传承中医智慧，守护全民健康****”为核心使命，通过智能健康数据采集、在线问诊、专业团队支持和科学知识普及，构建起“****预防—监测—干预—康复****”全周期健康闭环。用户不仅可以在家中完成基础健康指标记录，还能随时向专业中医师咨询，获取个性化的养生建议与治疗方案。**

**“仁医星中医馆”不仅是就医工具，更是个人健康的数字管家，让中医走进生活，让健康触手可及。**

## 二、核心功能
### 1. **首页功能入口（主界面）**
+ **减肥减脂****：针对肥胖、代谢综合征等问题提供专属健康管理路径。**
+ **中医鼻炎****：聚焦过敏性鼻炎、慢性鼻炎等常见病症，提供中医辨证施治方案。**
+ **其他疾病****：涵盖高血压、糖尿病、失眠、妇科调理等多种慢性病与亚健康问题。**
+ **服务团队****：展示医生、营养师、康复师等专业人员信息，增强信任感。**
+ **科普文章****：发布中医养生知识、节气调养指南、食疗推荐等内容，提升用户健康素养。**
+ **咨询****：一键发起在线咨询，连接专业医师进行病情沟通。**
+ **资讯****：推送最新活动公告、专家讲座、义诊通知等动态信息。**
+ **个人信息****：查看并编辑用户资料、检查报告、就诊记录等。**
+ **检查报告****：上传或查看体检结果，支持医生解读与趋势分析。**
+ **用户协议 / 投诉建议 / 退出登录****：保障使用规范与用户体验反馈渠道。**

---

### 2. **健康数据记录系统（手动录入模块）**
#### ✅ 记血糖
+ **用户可输入空腹/餐后血糖值（单位：mmol/L），选择检测时间。**
+ **支持连续记录，生成历史曲线图，帮助追踪血糖波动趋势。**
+ **适用于糖尿病患者、妊娠期高血糖人群等。**

#### ✅ 记血压
+ **输入收缩压（高压）、舒张压（低压）、心率数值（次/分）。**
+ **可标注测量时段（如晨起、睡前），便于医生判断血压变化规律。**
+ **适合高血压、心血管疾病患者日常监测。**

#### ✅ 记体型
+ **手动录入身高、体重、腰围、臀围等身体指标。**
+ **支持BMI计算与体脂评估，辅助体重管理和肥胖干预。**

#### ✅ 记饮食
+ **记录每餐类型（早餐/午餐/晚餐）、主食内容（如米饭、面条）。**
+ **支持添加饮食照片，直观反映进食结构。**
+ **助力饮食控制、控糖减脂、营养均衡管理。**

#### ✅ 记用药
+ **录入所服药品名称、剂量、服用时间、是否漏服。**
+ **防止错服、漏服，特别适合老年人或需长期服药的慢病患者。**

#### ✅ 记日常
+ **记录运动情况（如走路30分钟）、睡眠质量、身体感受（如“胃胀”“乏力”）。**
+ **实现生活方式的全面健康管理，助力中医“整体观”实践。**

---

### 3. **在线咨询服务（问医生）**
+ **用户可通过文字、语音或图片方式向专业中医师提问。**
+ **支持“在接咨询”状态显示，确保响应及时性。**
+ **医生可基于用户的健康档案（如血糖、血压、用药史）给出个性化建议。**
+ **提供“专家诊疗团队”入口，增强专业背书与服务可信度。**

---

### 4. **用户中心与信息管理**
+ **个人信息页****：**
    - **显示头像、昵称、性别、年龄、联系方式等基本信息。**
    - **支持修改账户设置、退出登录。**
+ **检查报告上传与管理****：**
    - **用户可上传体检报告、化验单等文件。**
    - **医生可在后台查看并出具专业解读意见。**
+ **资讯问答专区****：**
    - **提供常见问题解答，如“益生菌怎么吃？”、“茶包有没有副作用？”等，满足大众健康认知需求。**
    - **内容经过医学审核，确保科学准确。**

## 三、产品价值
### **用户价值：实现从“被动就医”到“主动健康管理”的转变**
+ **便捷高效****：无需跑医院，在家即可完成健康数据记录与在线问诊，节省时间和交通成本。**
+ **持续监测****：通过每日记录血糖、血压、饮食、运动等数据，形成完整的健康画像，有助于早发现异常。**
+ **个性化指导****：结合中医辨证理论，获得因人而异的调理建议，避免“千人一方”。**
+ **科学防病：通过饮食、作息、用药提醒，养成良好生活习惯，降低慢性病风险。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**
